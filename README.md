# The Guess

A luxury-inspired Number Guessing Game built using HTML, CSS, and JavaScript.

Designed with a premium editorial aesthetic inspired by high-end interior and lifestyle websites, this project combines elegant design with interactive gameplay.

## Overview

The application generates a random number between 1 and 100 and challenges the user to guess it. After each attempt, the game provides hints indicating whether the guess is too high or too low until the correct number is found.

## Features

* Random number generation
* Interactive guessing system
* Too High / Too Low hints
* Attempt counter
* Win notification
* Restart game functionality
* Input validation
* Fully responsive design
* Luxury editorial-inspired user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
